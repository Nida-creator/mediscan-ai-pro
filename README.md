# MediScan AI Pro

Pakistan's first AI-powered medical report explainer.

Upload any lab report and get a plain-language explanation in Urdu or English.

## Features
- 📄 Scan any lab report (PDF, JPG, PNG)
- 💬 AI Doctor Chat
- 📈 Health Trends
- 👪 Family Profiles
- 🌐 Urdu + English support

## Tech Stack
- AI: LLaMA 3.3 70B via Groq
- Backend: Flask
- Frontend: HTML, CSS, JavaScript
