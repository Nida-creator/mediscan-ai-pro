# 🩺 MediScan AI Pro

Pakistan's first AI-powered medical report explainer.

Upload any lab report and get a plain-language explanation in **Urdu or English** — free, instant, and always available.

## Features
- 📄 Scan any lab report (PDF, JPG, PNG)
- 💬 AI Doctor Chat — ask follow-up questions
- 📈 Health Trends — track values over time
- 👪 Family Profiles — manage your whole family
- 🌐 Urdu + English support

## Tech Stack
- **AI:** LLaMA 3.3 70B via Groq
- **Backend:** Flask
- **Frontend:** HTML, CSS, JavaScript
